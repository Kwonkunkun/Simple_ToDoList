## 개요
- html, css, javascript를 이용한 간단한 to do list

## 개발사항
- input text 입력 후 +버튼을 누르면 list에 올라감
- 한 일은 옆에 버튼 체크시 없어짐
- local storage에 저장하여 나갔다가 다시 들어가도 리스트에 저장되어있음

## 추가사항
